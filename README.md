# org-tree-demo

This is a demo of `org-tree-helper`, if you need more detail, see https://www.npmjs.com/package/org-tree-helper.

## Features

- 支持缩放和平移（使用 d3 实现）
- 支持父子节点之间跨层级连接
- 支持动态节点大小
- 支持横向与纵向显示
- 支持导出 SVG 文件
- 支持导出 PDF 文件

## Quick Overview

```sh
npm install
npm start
```
